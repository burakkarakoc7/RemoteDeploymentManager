# Contributing to Remote Deployment Manager

Projeye katkıda bulunmak istediğiniz için teşekkürler!

## Nasıl Katkı Sağlanır? (How to Contribute)

1. Bu depoyu (repository) fork edin.
2. Yeni bir branch oluşturun (`git checkout -b feature/AmazingFeature`).
3. Değişikliklerinizi yapın ve commit edin (`git commit -m 'Add some AmazingFeature'`).
4. Branch'inizi push edin (`git push origin feature/AmazingFeature`).
5. Bir Pull Request (PR) açın.

## Kod Standartları
* Python kodları **PEP 8** standartlarına uygun olmalıdır.
* Her yeni özellik için `tests/` klasörüne test eklenmelidir.
* Deployment scriptlerinde "Rollback" mekanizması bozulmamalıdır.
