# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 2.x     | :white_check_mark: |
| 1.x     | :x:                |

## Reporting a Vulnerability

Bu proje kritik sunucu altyapılarını yönettiği için güvenlik açıkları ciddiye alınır.

Eğer bir açık (vulnerability) bulursanız:
1. Lütfen GitHub Issues üzerinden **herkese açık paylaşmayın**.
2. Doğrudan `security@deployment-manager.com` adresine e-posta atın.
3. Ekibimiz 24 saat içinde dönüş yapacaktır.

**Kapsam:**
- SSH Key sızıntısı riski
- Remote Code Execution (RCE) açıkları
- Yetkisiz erişim (Privilege Escalation)
